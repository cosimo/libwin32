    This package contains all you need to get Activeware's Win32 Perl
    up and running with ODBC. It contains the essentials:

    + \lib\auto\Win32\ODBC.PLL          The ODBC extention for builds
                                        106 - 110.

    + \beta\lib\auto\Win32\ODBC.PLL     The ODBC extension for builds 303
                                        and higher.

    + \lib\Win32\ODBC.pm                The ODBC wrapper package.
    --------------------------------
    + README.TXT                        The instructions -- kinda'!    
    + TEST.PL                           ODBC extention test perl script.
    + ODBCTest.mdb                      Test Access database.
    + docs\*.htm                        Documentation in HTML format.

    This package was built using Win NT 4.0, Visual C++ 2.2 using Win32
    Perl build 110/303. This package was compiled for the 486 processor
    (I presume that most using this will be using at least a  486). All
    tests have been run successfully on my platform.
